# 🚀 Telecom X: Sistema Predictivo de Churn

Este proyecto forma parte de la estrategia de retención de clientes de **Telecom X**. El objetivo es utilizar técnicas de Machine Learning para identificar proactivamente qué clientes tienen una alta probabilidad de cancelar sus servicios y así reducir la tasa de evasión (Churn).

## 📋 Descripción del Proyecto
Tras un exhaustivo Análisis Exploratorio de Datos (EDA), detectamos que los clientes con contratos mensuales, cargos elevados y menos de un año de antigüedad son los segmentos de mayor riesgo. Este repositorio contiene el pipeline completo de modelado para transformar esos insights en predicciones accionables.

## 🛠️ Stack Tecnológico
* **Lenguaje**: Python 3.12
* **Procesamiento de Datos**: Pandas, NumPy
* **Visualización**: Matplotlib, Seaborn
* **Machine Learning**: Scikit-learn
* **Entorno**: Google Colab / Jupyter Notebooks
